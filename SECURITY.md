# Security Policy

## Supported Versions

Currently, this repository is a personal profile README. No specific versions are maintained for security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project or its external links (such as potential subdomain takeovers or malicious redirects), please do not report it publicly.

Instead, please send an email to the repository owner directly.

We will endeavor to review the issue and respond as quickly as possible.
